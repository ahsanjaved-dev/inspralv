# Genius365 - AI Voice Agent Platform

Genius365 is a white-label AI Voice Agent Management Platform. It enables agencies to manage AI voice agents across multiple providers (VAPI, Retell, Synthflow) with full multi-tenancy support, white-labeling, and workspace isolation.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Core Technologies

- **Framework**: Next.js 16 (App Router)
- **Database**: Supabase + Prisma ORM
- **Authentication**: Supabase Auth
- **UI**: Tailwind CSS 4 + Radix UI
- **State Management**: React Query + Zustand
- **Voice Providers**: VAPI, Retell, Synthflow
